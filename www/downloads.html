Simple Log Downloads


This page contains the most recent versions of Simple Log made available for download.

Previous releases of Simple Log are available from the project's Documents and files section .

Note that Simple Log requires J2SE version 1.4 or above to run.

Simple Log 2.0.1

Binaries

The full release of Simple Log contains the main distributable JAR, the rollover JAR, the adapter JARs, source code, user guide, API documentation and the fully-documented configuration files.

Full Release: simple-log-2.0.1.zip (985 K)


The Distributable JAR contains only the Simple Log classes, the Licence and the README.
The Rollover Distributable JAR contains the Simple Log classes needed for log rolling.

Distributable JAR Only: simple-log.jar (25.1 K)

Rollover Distributable JAR Only: simple-log-rollover.jar (11.4 K)

Adapter Binaries

The commons-logging JAR contains the CommonsLoggingAdapter class required to use Simple Log with Jakarta commons-logging.

The commons-logging adapter depends on the distributable JAR (simple-log.jar).

commons-logging Adapter JAR: simple-log-commons-logging.jar (11.2 K)


The SLF4J JAR contains an implentation of the SLF4J framework for using Simple Log through the SLF4J framework.

The SLF4J adapter depends on the distributable JAR (simple-log.jar).

SLF4J Adapter JAR: simple-log-slf4j.jar (22.1 K)
